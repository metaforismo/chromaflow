# Contributing

Thanks for helping improve ChromaFlow.

1. Open an issue for substantial behavior or schema changes.
2. Create a focused branch from `main`.
3. Keep image processing local and preserve preset schema compatibility.
4. Add tests for behavior changes and run `npm run check` plus `npm run test:e2e`.
5. Open a pull request explaining the problem, user impact, and observed validation.

Do not add analytics, image uploads, remote fonts beyond the framework-managed font integration, or third-party tracking without an explicit privacy review.

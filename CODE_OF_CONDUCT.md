# Contributor Covenant Code of Conduct

## Our pledge

We pledge to make participation in this project a harassment-free experience for everyone, regardless of age, body size, disability, ethnicity, sex characteristics, gender identity and expression, experience, education, socioeconomic status, nationality, appearance, race, caste, color, religion, or sexual identity and orientation.

## Our standards

Positive behavior includes empathy, constructive feedback, respect for differing viewpoints, and responsibility for mistakes. Unacceptable behavior includes harassment, sexualized language, personal attacks, trolling, and publishing private information without permission.

## Enforcement

Project maintainers may remove, edit, or reject contributions and communication that violate these standards. Report unacceptable behavior privately through the repository owner’s GitHub contact channels.

This code is adapted from Contributor Covenant 2.1.
